# Пельмень-Кликер

Idle-кликер для платформы Яндекс.Игры. Собирай пельмени, покупай бабушек,
строй пельмень-корпорацию, открывай скины и делай престиж на сметане.

## Механика

- 9 зданий-производителей (от «Бабушка Зина» до «Портал в Пельменную вселенную»)
- 7 улучшений клика (до ×5)
- 7 скинов с бонусом (до +50% дохода)
- Ачивки
- Престиж со сметаной (перманентный бонус)
- Rewarded video (Буст ×2 за просмотр рекламы)

## Стек

- HTML5 + чистый JavaScript (один файл)
- Yandex Games SDK v2
- Сохранения: SDK Player → localStorage → память
- ZIP < 10 КБ
